–°–ª–æ–∏ –æ–±—Ä–∞–∑–∞ –æ–ø–∏—Å—ã–≤–∞—é—Ç—Å—è –≤ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã—Ö —Ñ–∞–π–ª–∞—Ö –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏. –ö–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, –¥–ª—è —ç—Ç–æ–≥–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è Dockerfile. –ö–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–æ–Ω–Ω—ã–π —Ñ–∞–π–ª –≤—Å–µ–≥–¥–∞ –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å —É–∫–∞–∑–∞–Ω–∏—è –±–∞–∑–æ–≤–æ–≥–æ –æ–±—Ä–∞–∑–∞, –∏–º—è –∫–æ—Ç–æ—Ä–æ–≥–æ –ø—Ä–æ–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –ø–æ—Å–ª–µ –¥–∏—Ä–µ–∫—Ç–∏–≤—ã `FROM`. –î–∞–ª—å—à–µ –º–æ–≥—É—Ç –∏–¥—Ç–∏ —Ä–∞–∑–Ω—ã–µ –Ω–∞–¥—Å—Ç—Ä–æ–π–∫–∏ (–Ω–æ–≤—ã–µ —Å–ª–æ–∏) –æ–±—Ä–∞–∑–∞. –í—ã –º–æ–∂–µ—Ç–µ –∑–∞–¥–∞—Ç—å —Ä–∞–±–æ—á—É—é –ø–∞–ø–∫—É –ø—Ä–æ–µ–∫—Ç–∞ —Å –ø–æ–º–æ—â—å—é –¥–∏—Ä–µ–∫—Ç–∏–≤—ã `WORKDIR`, —Å–∫–æ–ø–∏—Ä–æ–≤–∞—Ç—å —Ñ–∞–π–ª—ã –≤ —ç—Ç—É —Ä–∞–±–æ—á—É—é –ø–∞–ø–∫—É –¥–∏—Ä–µ–∫—Ç–∏–≤–æ–π `COPY`, –∑–∞–ø—É—Å—Ç–∏—Ç—å –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –∫–æ–º–∞–Ω–¥—ã –∏–ª–∏ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –∫–æ–º–∞–Ω–¥ –≤ —Ç–µ—Ä–º–∏–Ω–∞–ª–µ –¥–∏—Ä–µ–∫—Ç–∏–≤–æ–π `RUN`. –ü—Ä–∏–º–µ—Ä –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏:

```dockerfile
FROM ubuntu:18.04

```

**–ö–æ–Ω—Ç–µ–π–Ω–µ—Ä (Docker Container)** ‚Äî —É–∂–µ —Å–æ–±—Ä–∞–Ω–Ω–æ–µ –∏ –∑–∞–ø—É—â–µ–Ω–Ω–æ–µ –ø—Ä–∏–ª–æ–∂–µ–Ω–∏–µ –≤ –∏–∑–æ–ª–∏—Ä–æ–≤–∞–Ω–Ω–æ–º –æ–∫—Ä—É–∂–µ–Ω–∏–∏, –∫–æ—Ç–æ—Ä–æ–µ —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç—Å—è –ø–æ—Å–ª–æ–π–Ω–æ, –≤ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–∏–∏ —Å –æ–±—Ä–∞–∑–æ–º. –ö–∞–∂–¥—ã–π –Ω–æ–≤—ã–π —Å–ª–æ–π —Ä–∞—Å—à–∏—Ä—è–µ—Ç —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª—å–Ω–æ—Å—Ç—å –ø—Ä–µ–¥—ã–¥—É—â–µ–≥–æ, —Ñ–æ—Ä–º–∏—Ä—É—è —Å—Ç–µ–∫ –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã—Ö –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–æ–≤, –ø–ª–∞—Ç—Ñ–æ—Ä–º –∏ –Ω–∞—Å—Ç—Ä–æ–µ–∫ —Å–∏—Å—Ç–µ–º–Ω—ã—Ö —Å–ª—É–∂–±. –§–∞–π–ª–æ–≤–∞—è —Å–∏—Å—Ç–µ–º–∞ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞ —Ç–æ–∂–µ —Å—Ç–µ–∫–æ–≤–∞—è ([Union File Systems](https://www.terriblecode.com/blog/how-docker-images-work-union-file-systems-for-dummies/)). –ö–∞—Ç–∞–ª–æ–≥–∏ –∏ —Ñ–∞–π–ª—ã –æ—Ç–¥–µ–ª—å–Ω–æ–≥–æ —Å–ª–æ—è –æ–±—Ä–∞–∑–∞ –Ω–∞–∫–ª–∞–¥—ã–≤–∞—é—Ç—Å—è –¥—Ä—É–≥ –Ω–∞ –¥—Ä—É–≥–∞, –æ–±—Ä–∞–∑—É—è –µ–¥–∏–Ω–æ–µ —Ü–µ–ª–æ–µ.

**–¢–æ–º (Docker Volume)** ‚Äî –ø–∞–ø–∫–∞, –∫–æ—Ç–æ—Ä—É—é –º–æ–∂–Ω–æ –ø–æ–¥–∫–ª—é—á–∏—Ç—å (–≥–æ–≤–æ—Ä—è—Ç _¬´–ø—Ä–∏–º–æ–Ω—Ç–∏—Ä–æ–≤–∞—Ç—å¬ª_) –∫ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞–º. –ü–∞–ø–∫–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å —Å–≤—è–∑–∞–Ω–∞ —Å –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–π –ø–∞–ø–∫–æ–π –Ω–∞ –≤–∞—à–µ–º –∫–æ–º–ø—å—é—Ç–µ—Ä–µ, –∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –∫–∞–∫ –±—ã —Å–µ—Ç–µ–≤–æ–π –¥–ª—è –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤ –Ω–∞ –≤–∞—à–µ–º –∫–æ–º–ø—å—é—Ç–µ—Ä–µ. –¢–æ–º–∞ –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã –¥–ª—è —Ö—Ä–∞–Ω–µ–Ω–∏—è —Ñ–∞–π–ª–æ–≤ –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏, –∫—Ä–∏—Ç–∏—á–µ—Å–∫–∏—Ö —Å —Ç–æ—á–∫–∏ –∑—Ä–µ–Ω–∏—è –±–µ–∑–æ–ø–∞—Å–Ω–æ—Å—Ç–∏, —Ñ–∞–π–ª–æ–≤ –±–∞–∑ –¥–∞–Ω–Ω—ã—Ö, —Ñ–∞–π–ª–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –Ω–µ–ª—å–∑—è —É–¥–∞–ª—è—Ç—å –ø–æ—Å–ª–µ –æ–∫–æ–Ω—á–∞–Ω–∏—è —Ä–∞–±–æ—Ç—ã –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è.

**–°–µ—Ç—å (Docker Network)** ‚Äî –≤–∏—Ä—Ç—É–∞–ª—å–Ω–∞—è –ª–æ–∫–∞–ª—å–Ω–∞—è —Å–µ—Ç—å, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å–æ–≤–º–µ—Å—Ç–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –∑–∞–ø—É—â–µ–Ω–Ω—ã—Ö –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤ –∏ —Å–æ–µ–¥–∏–Ω—è—Ç—å –∑–∞–ø—É—â–µ–Ω–Ω—ã–π –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä —Å –≤–∞—à–∏–º –∫–æ–º–ø—å—é—Ç–µ—Ä–æ–º. –í –æ—Å–Ω–æ–≤–Ω–æ–º –≤—ã –±—É–¥–µ—Ç–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç—Ä–∏ —Ä–µ–∂–∏–º–∞ —Ä–∞–±–æ—Ç—ã —Å–µ—Ç–µ–≤–æ–π –∏–Ω—Ñ—Ä–∞—Å—Ç—Ä—É–∫—Ç—É—Ä—ã Docker:

1. `bridge` ‚Äî –∫–æ–≥–¥–∞ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä—ã –º–æ–≥—É—Ç –≤–∑–∞–∏–º–æ–¥–µ–π—Å—Ç–≤–æ–≤–∞—Ç—å –º–µ–∂–¥—É —Å–æ–±–æ–π –∫–∞–∫ –≤–µ–±-—Å–µ—Ä–≤–µ—Ä –∏ –±–∞–∑–∞ –¥–∞–Ω–Ω—ã—Ö.
1. `host` ‚Äî –¥–ª—è –¥–æ—Å—Ç—É–ø–∞ –∫ –ª–æ–∫–∞–ª—å–Ω–æ–º—É —Å–µ—Ç–µ–≤–æ–º—É –æ–∫—Ä—É–∂–µ–Ω–∏—é –Ω–∞ –≤–∞—à–µ–º –∫–æ–º–ø—å—é—Ç–µ—Ä–µ.
1. `none` ‚Äî —Å–µ—Ç—å –¥–ª—è –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤ –ø–æ–ª–Ω–æ—Å—Ç—å—é –æ—Ç–∫–ª—é—á–µ–Ω–∞.


## –ö–∞–∫ –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è

–ö–ª—é—á–∏ –∫–æ–º–∞–Ω–¥–Ω–æ–≥–æ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–∞ Docker CLI —Ö–æ—Ä–æ—à–æ –ø—Ä–æ—Ä–∞–±–æ—Ç–∞–Ω—ã –∏ –ø–æ—Ö–æ–∂–∏ –Ω–∞ –∫–æ–Ω—Å–æ–ª—å–Ω—ã–µ –∫–æ–º–∞–Ω–¥—ã –≤ bash. –ù–∞–ø—Ä–∏–º–µ—Ä, –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã–π –∫–ª—é—á `prune` –ø–æ–∑–≤–æ–ª—è–µ—Ç —É–¥–∞–ª—è—Ç—å –Ω–µ–∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –æ–±—ä–µ–∫—Ç—ã. –ö–ª—é—á `rm` —Å–ª—É–∂–∏—Ç –¥–ª—è —É–¥–∞–ª–µ–Ω–∏—è, –∞ –∫–ª—é—á `ls` –¥–ª—è –ø—Ä–æ—Å–º–æ—Ç—Ä–∞ –æ–±—ä–µ–∫—Ç–æ–≤. –û–±—ä–µ–∫—Ç—ã Docker –≤ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ–º –ø–æ—Ä—è–¥–∫–µ –∏–º–µ—é—Ç —É–Ω–∏–∫–∞–ª—å–Ω–æ–µ –∏–º—è. –ï—Å–ª–∏ –≤—ã –Ω–µ –∏–º–µ–Ω—É–µ—Ç–µ –æ–±—ä–µ–∫—Ç —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ, —Ç–æ –∏–º—è –æ–±—ä–µ–∫—Ç–∞ —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç—Å—è —Å –ø–æ–º–æ—â—å—é —Ö—ç—à-—Ñ—É–Ω–∫—Ü–∏–∏. –ï—Å–ª–∏ –≤—ã –ø–æ–ø—ã—Ç–∞–µ—Ç–µ—Å—å —Å–æ–∑–¥–∞—Ç—å –æ–±—ä–µ–∫—Ç –æ–¥–Ω–æ–≥–æ –∏ —Ç–æ–≥–æ –∂–µ —Ç–∏–ø–∞ —Å —É–∂–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–Ω—ã–º –∏–º–µ–Ω–µ–º, –≤ —ç—Ç–æ–º –≤–∞–º –±—É–¥–µ—Ç –æ—Ç–∫–∞–∑–∞–Ω–æ. –ö–∞–∫ –∂–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∫–æ–Ω—Å–æ–ª—å–Ω—ã–º –∫–ª–∏–µ–Ω—Ç–æ–º?

### –ú–æ–Ω–∏—Ç–æ—Ä–∏–Ω–≥ –∑–∞–ø—É—â–µ–Ω–Ω—ã—Ö –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤

- `docker ps` ‚Äî –ø—Ä–æ—Å–º–æ—Ç—Ä –∑–∞–ø—É—â–µ–Ω–Ω—ã—Ö –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤.
- `docker ps -a` ‚Äî –∫–ª—é—á `-a` –≤—ã–≤–æ–¥–∏—Ç –∏ –∑–∞–ø—É—â–µ–Ω–Ω—ã–µ, –∏ –æ—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–Ω—ã–µ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä—ã.
- `docker ps -s` ‚Äî –∫–ª—é—á `-s` –≤—ã–≤–æ–¥–∏—Ç –¥–∏—Å–∫–æ–≤–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ, –∏—Å–ø–æ–ª—å–∑—É–µ–º–æ–µ –∫–∞–∂–¥—ã–º –∑–∞–ø—É—â–µ–Ω–Ω—ã–º –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–º.

–ü–æ–ª–Ω—ã–π —Å–ø–∏—Å–æ–∫ –∫–ª—é—á–µ–π –¥–ª—è –∫–æ–º–∞–Ω–¥—ã `docker ps` –¥–æ—Å—Ç—É–ø–µ–Ω [–≤ –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–∏](https://docs.docker.com/engine/reference/commandline/ps/).

### –ó–∞–ø—É—Å–∫ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–æ–≤

–î–ª—è –∑–∞–ø—É—Å–∫–∞ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞, –∫–æ—Ç–æ—Ä—ã–π –¥–æ—Å—Ç—É–ø–µ–Ω –ª–æ–∫–∞–ª—å–Ω–æ –∏–ª–∏ –Ω–∞ Docker Hub, –≤—ã–ø–æ–ª–Ω–∏—Ç–µ –∫–æ–º–∞–Ω–¥—É:

–ö–ª—é—á `--name` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —É—Å—Ç–∞–Ω–æ–≤–∫–∏ –∏–º–µ–Ω–∏ –∑–∞–ø—É—â–µ–Ω–Ω–æ–≥–æ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞. –ö–ª—é—á–∏ `-i` –∏ `-t` —É–∫–∞–∑—ã–≤–∞—é—Ç, —á—Ç–æ –¥–ª—è –∑–∞–ø—É—Å–∫–∞ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞ –±—É–¥–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è [—Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–π –ø–æ—Ç–æ–∫ –≤–≤–æ–¥–∞](https://ru.wikipedia.org/wiki/–°—Ç–∞–Ω–¥–∞—Ä—Ç–Ω—ã–µ_–ø–æ—Ç–æ–∫–∏) –∏ [—Ç–µ—Ä–º–∏–Ω–∞–ª TTY](https://ru.wikipedia.org/wiki/TTY-–∞–±—Å—Ç—Ä–∞–∫—Ü–∏—è) —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ. –î–ª—è —Ç–æ–≥–æ —á—Ç–æ–±—ã –ø—Ä–∏ –∑–∞–ø—É—Å–∫–µ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä–∞ –ø—Ä–∏–º–æ–Ω—Ç–∏—Ä–æ–≤–∞—Ç—å —Ç–æ–º, –∫–æ—Ç–æ—Ä—ã–π –±—É–¥–µ—Ç —Å–≤—è–∑–∞–Ω —Å –ø–∞–ø–∫–æ–π –Ω–∞ –≤–∞—à–µ–º –∫–æ–º–ø—å—é—Ç–µ—Ä–µ, –∞ –ø–æ—Ç–æ–º –ø–æ–ª—É—á–∏—Ç—å –¥–æ—Å—Ç—É–ø –∫ –∫–æ–Ω—Ç–µ–π–Ω–µ—Ä—É —á–µ—Ä–µ–∑ —Ç–µ—Ä–º–∏–Ω–∞–ª, –≤—ã–ø–æ–ª–Ω–∏—Ç–µ –∫–æ–º–∞–Ω–¥—É:

–ü–æ–ª–Ω—ã–π —Å–ø–∏—Å–æ–∫ –∫–ª—é—á–µ–π –¥–ª—è –∫–æ–º–∞–Ω–¥—ã `docker run` –¥–æ—Å—Ç—É–ø–µ–Ω [–≤ –¥–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏–∏](https://docs.docker.com/engine/reference/commandline/run/).

### –£–ø—Ä–∞–≤–ª–µ–Ω–∏–µ –æ–±—Ä–∞–∑–∞–º–∏

–í—ã –º–æ–∂–µ—Ç–µ –ø–æ–ª—É—á–∏—Ç—å —Å–ø–∏—Å–æ–∫ –≤—Å–µ—Ö –¥–æ—Å—Ç—É–ø–Ω—ã—Ö –ª–æ–∫–∞–ª—å–Ω–æ –æ–±—Ä–∞–∑–æ–≤ —Å –ø–æ–º–æ—â—å—é –∫–æ–º–∞–Ω–¥—ã:

–ö–ª—é—á–∏ `prune`, `rm` –¥–µ–π—Å—Ç–≤—É—é—Ç –æ–±—ã—á–Ω—ã–º —Å–ø–æ—Å–æ–±–æ–º, –ø–æ–∑–≤–æ–ª—è—è —É–¥–∞–ª–∏—Ç—å –Ω–µ–∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ –∏–ª–∏ –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã–µ –æ–±—Ä–∞–∑—ã —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ. –î–ª—è —Ä–∞–±–æ—Ç—ã —Å —Ä–µ–µ—Å—Ç—Ä–æ–º –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–ª–µ–¥—É—é—â–∏–µ –∫–æ–º–∞–Ω–¥—ã:
